Изменения в версии 1.19
===========================

Эта версия содержит поддержку платы контроллера CMS SCSI, совместимого со всеми моделями Apple II.
Модуль эмуляции контроллера позволяет подключить до 3 виртуальных жестких дисков с указанием их идентификаторов шины.
Каждый диск, размер которого больше чем 32 МБ (максимальный размер тома ProDOS) будет разбит на несколько логических разделов.
После загрузки, ProDOS будут доступны только два логических раздела.
Чтобы указать, какие разделы будут доступны для ProDOS, при загрузке системы необходимо нажать левую клавишу Alt, как это было в случае с оригинальной картой контроллера.
Эта версия эмулятора содержит пример 64МБ SCSI диска, на котором находится ProDOS, Basic и необходимые утилиты для карты CMS.
Второй логический раздел этого диска отформатирован как пустой том /PART2.

Чтобы создать новый образ диска SCSI, достаточно в интерфейсе настройки контроллера указать имя несуществующего файла, затем отформатировать новое устройство с помощью программы CMS.SYSTEM (низкоуровневое форматирование), а затем создать новый том ProDOS с помощью программы FILER.

Удачи!
