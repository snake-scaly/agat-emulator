Для сборки новой версии дистрибутива необходимо:
1. Обновить номер версии в формате major.minor[.revision] (далее {ver}) в файле version.h в основном дереве исходных текстов
2. Выполнить скрипт makeall.bat в основном дереве исходных текстов
3. Выполнить скрипт release.bat в основном дереве исходных текстов
4. Выполнить скрипт make.bat в каталоге help дерева дистрибутива предварительно проверив путь к компилятору hhc
5. Подготовить файлы release-ver-en.txt и release-{ver}-ru.txt в дереве дистрибутива
6. Выполнить make.bat {ver} в каталоге setup дерева дистрибутива предварительно проверив путь к компилятору iscc 5.x
7. Полученный файл agatemulator-{ver}.exe из каталога trunk дерева дистрибутива закачать на страницу проекта с помощью скрипта Upload.bat
8. Выставить в веб-интерфейсе проекта текущую версию для скачивания
