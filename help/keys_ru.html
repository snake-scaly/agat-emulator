Клавиатурные сокращения

В эмуляторе используются следующие клавиатурные сокращения для работающей системы:

Key
Action
Ctrl+Break Сброс (Упр+Сбр)
Alt+Break Аппаратный сброс (выкл\вкл)
F1...F3 Функциональные клавиши Агата
F4 Переключение режима цвета монитора
F5 Разблокировать мышь
F6 Перенаправить ввод из текстового файла
F7 Копировать текущий экран в буфер обмена
F8 Запустить/завершить отладчик (если он включен в настройках эмулятора)
Alt+Enter Переключение полноэкранного режима
F10 Вход в контекстное меню
Keypad keys Функциональные клавиши Агата
Windows App key Включение "быстрого" режима
Alt+F4 Закрытие окна текущей системы
