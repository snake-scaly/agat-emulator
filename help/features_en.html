Main features of emulator

  1. Full support of Apple I, Apple ][, Apple ][ Plus, Apple //e and Enhanced Apple //e;
  2. Full support of Agat-7, Agat-9 systems;
  3. Modular fully configurable architecture;
  4. Support for multiple systems configurations;
  5. Support for floppy images in both sector and raw (nibble) formats including write support;
  6. Support for extended hardware such as Z80 Softcard (fuse), Videx Videoterm 80-column card, Thunderclock and Mockingboard II;
  7. Three configurable CPU modules: 6502 (developed with this project), M6502 (by Marat Fayzullin and Alex Krasivsky) and 65C02;
  8. Undocumented CPU commands support in 6502 module;
  9. Fully configurable memory sizes (4K...48K for base memory);
  10. Joystick support;
  11. Full tape support via Microsoft .WAV files;
  12. Both color and mono monitors support;
  13. Support for DirectSound, hardware PC Speaker and Waveform Audio for beeper emulation;
  14. Selectable ROM images and fonts;
  15. Built-in Epson FX printer emulation;
  16. Support for combined video-modes via timer interrupts for Agats;
  17. Support for Apple ][ Memory Expansion Cards, RamFactor (with RamCharger) Cards, Saturn 16/32/64/128K Memory cards;
  18. Support for SCSI controller;
  19. Support for Liberty Drive controller for Apple //e / Pravetz 8a;
  20. Support for Dallas SmartWatch No-Slot-Clock for Apple //e / Pravetz 8a;
  21. Support for Apple ][ Firmware card;
  22. Support for custom keymaps and screen palettes.
