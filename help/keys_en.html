Keyboard shortcuts

Key
Action
Ctrl+Break Reset
Alt+Break Hard reset
F1...F3 Agat functional keys
F4 Toggle Color/Mono in Apple ][ mode
F5 Unlock mouse (if it is captured)
F6 Redirect input from text file
F7 Copy current screen to clipboard
F8 Start debugger (if it is enabled).
Alt+Enter Toggle fullscreen mode
F10 Enter context menu
Keypad keys Agat functional keys
Windows App key Enter "fast" mode
Alt+F4 Close current window
